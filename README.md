## Assignment #3
### 원형 이중연결리스트 이용, 가위바위보 프로그램 작성
- 사용자가 이기면 win +1 & 오른쪽 노드 이동, 지면 lose +1 & 왼쪽 노드로 이동
- win >= 2이면 win 0으로 변경 후 현재 노드 양쪽에 새로운 노드 삽입
- lose >= 2이면 현재 노드 삭제
- 노드 0개 되면 종료
